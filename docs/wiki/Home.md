# OrientaCar

**Inteligência a favor do produtor rural.**
Solução desenvolvida para o haCARthon 2026 — Desafio 1: Simplificar o CAR para o usuário.

O OrientaCar é um aplicativo mobile que guia produtores rurais no preenchimento e correção do Cadastro Ambiental Rural passo a passo, usando inteligência artificial e dados públicos para analisar cada etapa antes do envio e traduzir erros em instruções simples e acessíveis.
