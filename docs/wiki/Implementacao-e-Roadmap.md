# Implementação e Roadmap

A estimativa é de 6 meses para uma versão de produção completa, dividida em três fases.

## Fase 1 — 48 horas ✅ Concluída no haCARthon

O protótipo funcional foi entregue durante o evento com todas as telas principais implementadas, cadastro guiado completo em 7 etapas, chat IA com respostas pré-configuradas, validação pré-envio com alertas e bloqueios, e salvamento automático de rascunhos. Os dados de integração são mock — simulações que representam o comportamento real das APIs do governo.

## Fase 2 — 3 meses

Substituir os dados mock por integrações reais com as APIs do SICAR, MapBiomas e Gov.br. Realizar testes de usabilidade com produtores rurais reais no Tocantins. Expandir o banco de conhecimento do Chat IA com base na legislação do CAR, cobrindo os erros mais comuns identificados pelos analistas das OEMAs. Ajustar a experiência a partir do que os testes revelarem.

## Fase 3 — 3 meses

Lançar o bot de WhatsApp para envio de alertas de prazo e orientações por mensagem — o canal preferido do produtor rural. Adicionar mapeamento por GPS caminhando pelo perímetro da propriedade. Publicar um painel público de impacto com métricas de cadastros assistidos e erros evitados. Iniciar a expansão para outros estados.

## O que será necessário

As integrações técnicas exigem acesso às APIs públicas do SICAR, MapBiomas, PRODES e Gov.br, além de integração com SIGEF/INCRA para dados de imóveis rurais e com o ONR para dados cartoriais.

A validação de conteúdo passa por especialistas em legislação ambiental, analistas das OEMAs para mapeamento dos erros mais frequentes, e testes com produtores rurais de diferentes perfis.

A distribuição será feita como PWA via QR code nos órgãos ambientais estaduais e sindicatos rurais, com treinamento dos analistas para uso do painel de acompanhamento.
