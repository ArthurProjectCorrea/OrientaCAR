## Linha do tempo

**6 meses** para uma versão de produção completa, em três fases.

---

### Fase 1 — 48 horas ✅ Concluída no haCARthon

Protótipo funcional com:
- Todas as telas principais implementadas
- Dados mock para todas as integrações
- Cadastro guiado completo em 7 etapas
- Chat IA com respostas pré-configuradas
- Validação pré-envio com alertas e bloqueios
- Salvamento automático de rascunhos

---

### Fase 2 — 3 meses

- Integração real com as APIs do SICAR, MapBiomas e Gov.br (substituindo dados mock)
- Testes de usabilidade com produtores rurais reais no Tocantins
- Expansão do banco de conhecimento do Chat IA com base na legislação do CAR
- Ajustes de experiência a partir dos testes

---

### Fase 3 — 3 meses

- Bot WhatsApp para alertas de prazo e orientações por mensagem
- Mapeamento por GPS caminhando pelo perímetro da propriedade
- Painel público de impacto (quantidade de cadastros assistidos, erros evitados)
- Expansão para outros estados

---

## O que seria necessário

### Integrações técnicas

1. APIs públicas do SICAR, MapBiomas, PRODES e Gov.br
2. SIGEF/INCRA para dados de imóveis rurais
3. ONR (Operador Nacional do Registro) para dados cartoriais

### Validações de conteúdo

1. Especialistas em legislação ambiental e Código Florestal
2. Analistas das OEMAs para mapeamento dos erros mais frequentes
3. Testes de usabilidade com produtores rurais de diferentes perfis

### Distribuição

1. Publicação como PWA
2. Distribuição via QR code nos órgãos ambientais estaduais e sindicatos rurais
3. Treinamento dos analistas para uso do painel de acompanhamento
