# Como Contribuir

Obrigado pelo interesse em contribuir com o OrientaCar! Este projeto nasceu no haCARthon 2026 com o objetivo de se tornar um Bem Público Digital — qualquer contribuição que ajude produtores rurais a regularizar sua situação ambiental é bem-vinda.

## Antes de começar

- Leia o [README](README.md) para entender o projeto
- Veja as [issues abertas](https://github.com/ArthurProjectCorrea/OrientaCAR/issues) para evitar trabalho duplicado
- Para mudanças grandes, abra uma issue primeiro para alinhar a direção

## Como enviar uma contribuição

1. Faça um fork do repositório
2. Crie uma branch para sua alteração:
   ```
   git checkout -b feature/nome-da-funcionalidade
   ```
3. Faça suas alterações e commite seguindo o padrão abaixo
4. Abra um Pull Request descrevendo o que foi feito e por quê

## Padrão de commits

Use o formato `tipo: descrição curta em português`. Exemplos:

```
feat: adiciona validação de CPF no formulário
fix: corrige cálculo de área no mapa
docs: atualiza instruções de instalação
style: ajusta espaçamento na tela de login
refactor: simplifica lógica de salvamento de rascunho
```

## Reportar um problema

Use o template de [bug report](https://github.com/ArthurProjectCorrea/OrientaCAR/issues/new?template=bug_report.md) e inclua:
- O que você esperava que acontecesse
- O que aconteceu de verdade
- Passos para reproduzir o problema

## Sugerir uma melhoria

Use o template de [feature request](https://github.com/ArthurProjectCorrea/OrientaCAR/issues/new?template=feature_request.md) e descreva o problema que a melhoria resolveria.

## Código de conduta

Este projeto segue o [Código de Conduta](CODE_OF_CONDUCT.md). Ao contribuir, você concorda em respeitá-lo.
